# Portfolio Website - Aryan Das

This repository contains the source code for the professional portfolio website of Aryan Das. The project is designed with a premium, minimalist aesthetic, featuring a deck-based navigation system and custom interactive elements.

## Project Overview

The portfolio serves as a comprehensive digital showcase of skills, certifications, and professional background. It emphasizes a clean user interface, smooth transitions, and high-performance vanilla JavaScript interactions.

## Key Features

### Deck-Mode Navigation
The website utilizes a vertical scroll-snap system that divides content into structured slides or "decks." This is complemented by:
- Side navigation dots for direct section access.
- Keyboard support (Arrow keys, PageUp/PageDown) for improved accessibility.
- Smooth eased scrolling for a refined user experience.

### Responsive Micro-Animations
A custom intersection observer system triggers reveal animations as components enter the viewport. This ensures that content is presented dynamically without overwhelming the user.

### Interactive Skills Display
An advanced "Skills Burst" overlay allows users to explore the technical stack through a randomly distributed, animated cloud of skill tags. This feature is implemented using vanilla JavaScript for optimal performance.

### Mobile Optimization
The layout and navigation systems are fully responsive, adapting fluidly from large desktop displays to mobile devices.

## Technology Stack

The project is built using modern web standards without reliance on external frameworks, ensuring fast load times and high maintainability:
- **HTML5**: Semantic structure for accessibility and SEO.
- **CSS3**: Custom layout systems, glassmorphism effects, and CSS variables for consistent design tokens.
- **JavaScript (ES6+)**: Vanilla implementation for all interactive logic, including the navigation engine and animation observers.

## Project Structure

- `index.html`: The primary landing page containing the main deck structure.
- `skills.html`: Detailed inventory of technical skills and certifications.
- `contact.html`: Dedicated contact profile and informational page.
- `styles.css`: Comprehensive design system and layout configurations.
- `site.js`: Core engine for navigation, animations, and interactive overlays.
- `favicon.png`: Custom monogram branding for the application.

## Local Development

To run this project locally, clone the repository and open the `index.html` file in any modern web browser. 

For development with hot-reloading:
1. Ensure you have a local server environment (e.g., Live Server extension for VS Code or any HTTP server).
2. Start the server from the root directory.
